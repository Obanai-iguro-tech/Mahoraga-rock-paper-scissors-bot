#  Rock Paper Scissors Mahoraga Bot 

A smart rock-paper-scissors bot that learns and adapts based on your playing patterns. Like Mahoraga from Jujutsu Kaisen the more you play, the more it adapts!

## Features
-Adaptive AI: The bot uses pattern recognition to predict your next move
-Learning System: Analyzes your game history to improve predictions
-Score Tracking: Keeps track of wins, losses, and ties
-Data: Game history is saved between sessions

## How It Works
The bot uses a pattern prediction algorithm that:
1. Looks at the last 3 moves in your history
2. Finds similar patterns from previous games
3. Predicts what you'll play next
4. Counters your predicted move

After 10 games the bot has enough data to start making accurate predictions
